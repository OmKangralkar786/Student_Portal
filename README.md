# Student Learning Portal

A Django-based Student Learning Portal that allows students and teachers to manage courses, study materials, notes, and learning resources.

## Features

- Student Login & Registration
- Teacher Login
- Course Management
- Study Materials Upload
- Notes Management
- Semester-wise Courses
- Responsive UI

## Technologies Used

- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/student-portal.git
cd student-portal

pip install -r requirements.txt

python manage.py migrate
python manage.py runserver
